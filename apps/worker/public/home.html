Quick Start

Deploy HTML, Markdown, or code files instantly with the sharehtml CLI:

npx -y @duyet/sharehtml@latest deploy path/to/file.html
{{#requires_login}}

First, authenticate with your instance:

npx -y @duyet/sharehtml@latest login
{{/requires_login}}

Add sharehtml skills to your AI agent:

npx -y skills add duyet/sharehtml

Or paste this prompt to your AI agent: 

"Deploy this to the web using sharehtml: `npx -y @duyet/sharehtml deploy path/to/file.html`"

The agent will deploy files and return the live URL automatically.
