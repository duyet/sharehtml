Quick Start

Deploy HTML, Markdown, or code files instantly with the sharehtml CLI:

npx -y @duyet/sharehtml@latest deploy path/to/file.html
{{#requires_login}}

First, authenticate with your instance:

For AI Agents: Share your screen with Claude Code and other AI agents:

npx -y @duyet/sharehtml@latest deploy file.html --prompt="Review this file with me"
